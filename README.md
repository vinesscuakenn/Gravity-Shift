# Gravity Shift

An innovative game built with Python and Pygame where you control a ball that can switch gravity directions to navigate a hazardous environment. Collect yellow gems to increase your score while avoiding red spikes within a 30-second time limit.

## Installation

1. Install Python 3.x from [python.org](https://www.python.org/).
2. Install Pygame by running `pip install pygame` in your terminal.

## How to Play

1. Run the `gravity_shift.py` file using Python.
2. Use the **Arrow Keys** to move the ball (white circle) around the screen.
3. Press the **Spacebar** to cycle through gravity directions (down, up, left, right).
4. Collect yellow gems to increase your score.
5. Avoid red spikes, which reduce your health.
6. Survive for 30 seconds or until you lose all 3 health points.
7. Your score is based on the number of gems collected.

## Controls

- **Left Arrow**: Move left
- **Right Arrow**: Move right
- **Up Arrow**: Move up
- **Down Arrow**: Move down
- **Spacebar**: Switch gravity direction
